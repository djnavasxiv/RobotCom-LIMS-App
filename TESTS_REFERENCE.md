### Pages for Test Results
All test results are entered through the "Resultados" (Test Results) page:
1. Select a pending sample (Muestra)
2. Choose the test type (Pruebas)
3. Enter the individual test results
4. Generate and save the report

---

## Test Entry Process

### Steps to Enter Test Results:

1. **Navigate to Resultados (Test Results)** page from main menu
2. **Select a Sample (Muestra)** from the list of pending samples
3. **View Pending Exams** - System shows which tests need results
4. **Select Test Type** - Click on one of the 9 test categories
5. **Enter Results** - Input individual test values and reference ranges
6. **Validate Data** - System checks for valid ranges
7. **Save Results** - Store results in the database
8. **Generate Report** - Create printable/shareable report

---

- All test categories support input validation

- Results can be entered in Spanish or English
- All data is encrypted and securely stored
- Reports can be printed or exported as PDF
